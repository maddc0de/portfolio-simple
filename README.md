# My personal portfolio
Built with plain HTML, CSS, and TypeScript.

## Prerequisites
- [Node.js](https://nodejs.org/) (includes npm and npx)

## Running locally in VS Code
1. Clone the repo
2. Install the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension in VS Code
3. Right-click `index.html` → **Open with Live Server**
4. Opens automatically at `http://127.0.0.1:5500`

## Making changes to TypeScript
`main.js` is already compiled and committed. If you edit `main.ts`, 
recompile with:
```bash
npx tsc --watch
```

## Tech
- HTML5
- CSS3 (custom properties for theming)
- TypeScript
